# Registar
HTML indeks za registrovane unose, medijske tragove i zapise u zatvorenoj proceduri

Vidljivi indeks služi za pristup unosima.  
`REGISTAR_LEDGER.csv` služi kao evidencija zatvaranja.

## Struktura

- `index.html` — javni indeks
- `style.css` — vizuelni sistem
- `entries/` — pojedinačni unosi
- `media/` — slike, audio i video fajlovi
- `REGISTAR_LEDGER.csv` — evidencija zapisa
- `tools/` — lokalne skripte za zatvaranje i proveru

- ## Pravilo

Status zapisa može biti zatvoren.  
Status događaja može ostati nerazrešen.
